# Modern Analytics Table UI

A modern analytics-style table interface built using HTML and CSS.  
This project demonstrates how to create visually appealing data tables with gradient backgrounds and clean UI styling.

## Technologies Used
- HTML5
- CSS3

## Features
- Modern table UI design
- Gradient background
- Clean and readable data layout
- Styled table rows and cells
- Dashboard-style interface

## Purpose of the Project
This project was created as part of my front-end development practice to improve my skills in styling HTML tables and building modern UI layouts using CSS.
